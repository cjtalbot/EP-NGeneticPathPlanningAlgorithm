FindAndLaunchCanvasJSGS()
Helper function which is automatically called to launch JSGameSoup on a canvas. Finds all canvas tags in the document, calls the function named in the attribute 'jsgs', passes a new JSGameSoup instance to that function, fps is set in the attribute called 'fps'. Modified version of processingjs' init.js example script.
Defined in: jsgamesoup.js.
